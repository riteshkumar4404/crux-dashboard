# crux-dashboard
A dashboard for displaying matrix for the URL

## Instructions to run:

Open terminal
Use Node version: v20.17.0
`cd client`
`npm install`
`npm start`

Use Node version: v20.17.0
Open terminal

`cd server`
`npm install`
`npm run dev`
